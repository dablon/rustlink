¿Por qué RustLink?

La alternativa descentralizada a los mensajes tradicionales

🌐
P2P Descentralizado
Sin servidores centrais. La comunicación es peer-to-peer usando libp2p. Tu datos, tu control.
🔐
Cifrado End-to-End
Mensajes cifrados con claves Ed25519. Solo tú y tu destinatario pueden leer.
📱
CLI Minimalista
Interfaz rápida y ligera. Funciona en cualquier terminal. Sin GUI innecesaria.
📦
Transferencia de Archivos
Comparte archivos directamente con tus amigos, sin intermediarios.
🌍
Código Abierto
100% gratuito y de código abierto. Audita el código tú mismo.
🔄
Multi-Plataforma
Funciona en Linux, macOS, y Windows. Una experiencia unificada.

🚀 Quick Start

terminal 
# Instalar desde código fuente
cargo install rustlink

# O descargar binario
curl -L https://github.com/dablon/rustlink/releases/latest -o rustlink

# Inicializar identidad
rustlink init tu_usuario

# Ver estado
rustlink status

Elige tu método preferido
